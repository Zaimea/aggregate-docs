## Instalation

You can install the package via composer:

```bash
composer require zaimealabs/aggregate
```

or via composer.json

```json
"repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/zaimealabs/aggregate"
        }
    ]
```
